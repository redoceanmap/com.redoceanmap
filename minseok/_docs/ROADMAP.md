- 뉴스 감성 피처의 백테스트 투입은 **약 3개월 축적 후** (표본 기준 미달 전엔 확률 주장 금지, 참고 정보로만)

- 자체 모델 경로: 유해분류 v1(완료) → v2(운영 오탐 반영) → 뉴스 감성 한국어 경량 모델(KcELECTRA 파인튜닝). 자체 LLM 사전학습은 금지
- **후보(2026-07-14 추가): 상권 뉴스 수집 + RAG** — 지역 상권·정책·재개발 기사를 수집해
  chat 상권 답변에 기사 근거를 결합("성수동 요즘 분위기 어때?"). 주식 뉴스 RAG 패턴
  (수집 cron → 허브 적재 → bge-m3 임베딩 → 검색 포트) 그대로 복제 — 수치(공공데이터)는 포트
  조회, 텍스트는 RAG 원칙 유지. **①-M3(스코어링) 이후 검토** — 수집원(지역지 RSS 등) 조사가 선행 조건
