# 🗺️ RedOcean Map — 서울 상권 분석 서비스

> 창업을 고민하는 사람들을 위한 데이터 기반 상권 분석 플랫폼

---

## 왜 만들게 됐나

요즘 창업을 생각하는 사람들은 많은데, 막상 "어디서 하면 좋을까?"를 제대로 알려주는 서비스가 없다.
포털 지도에서 주변 카페 수를 세거나, 유튜브 창업 브이로그를 보거나, 발품 팔아 돌아다니거나.

서울시에는 상권분석 공공데이터가 있다. 유동인구, 추정매출, 폐업률, 소비패턴까지.
근데 이 데이터가 CSV 파일로 제공되고 있어서 일반인이 쓰기엔 너무 불편하다.

그래서 직접 만들기로 했다.
공공데이터를 파싱해서 DB에 쌓고, 지도 위에 올리고, AI가 분석해주는 서비스.
이름은 **RedOcean Map** — 레드오션인 줄 알고 포기하려다가, 데이터를 보면 의외로 블루오션인 곳을 찾아주는 서비스라는 의미다.

---

## 어떤 서비스인가

```
사용자: "강남 근처에서 카페 창업하고 싶어요"
     ↓
AI가 서울 1,577개 상권 데이터를 분석
     ↓
"강남역 상권은 월 점포당 평균 매출 4,800만원이지만
 폐업률이 분기 6%로 높습니다. 대신 논현동 상권은..."
```

AI가 추정하는 게 아니라, 서울시 공공데이터 실수치를 직접 계산해서 알려준다.

---

## 기술 스택

| 영역 | 기술 |
|------|------|
| 백엔드 | Python, FastAPI, SQLAlchemy (async) |
| DB | PostgreSQL (클라우드 호스팅) |
| 프론트엔드 | Next.js, TypeScript, TailwindCSS |
| AI | Google Gemini API |
| 데이터 | 서울시 공공데이터 (상권분석서비스) |

---

## 데이터 규모

서울시 상권분석서비스 기준:
- **상권 수**: 1,577개
- **테이블**: 9개 (추정매출, 점포, 유동인구, 상주인구, 직장인구, 소비, 아파트, 상권변화, 영역)
- **분기별 업데이트**: 매 분기 신규 데이터 적재
*계속해서 더 많은 데이터, 서비스를 추가할 예정이다*
